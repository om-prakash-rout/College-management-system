


# College Management System (Frontend)

## Description
This is a frontend project for managing college-related information.  
It provides a user-friendly interface to view and manage students, faculty, courses, attendance, and marks.  
The system is purely frontend, built with HTML, CSS, and JavaScript.  

## Features
- Responsive UI using HTML, CSS, and JavaScript
- Separate pages for:
  - Student Information
  - Faculty Information
  - Courses & Subjects
  - Marks & Grades
  - Attendance
- Interactive tables and forms
- Search and filter options for easy data lookup
- Professional and clean layout for desktop and mobile
